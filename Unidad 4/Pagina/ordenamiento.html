Algoritmos de Ordenamiento

Los algoritmos de ordenamiento son métodos sistemáticos para organizar los elementos de una lista o conjunto de datos en un cierto orden. El ordenamiento es un problema común en ciencias de la computación y se utiliza para mejorar la eficiencia de búsquedas y operaciones en conjuntos de datos.






Tipos de Algoritmos de Ordenamiento


Burbuja (Bubble Sort)

Compara pares de elementos adyacentes y los intercambia si están en el orden incorrecto. Este proceso se repite hasta que la lista esté ordenada. Es simple pero no eficiente para listas grandes.


Selección (Selection Sort)

Encuentra el elemento mínimo en la lista y lo intercambia con el primer elemento. Luego, encuentra el mínimo en el resto de la lista y lo intercambia con el segundo elemento, y así sucesivamente. No es eficiente para grandes conjuntos de datos, pero puede ser más eficiente en comparación con la burbuja.


Inserción (Insertion Sort)

Construye una secuencia ordenada de elementos uno a la vez tomando elementos de la lista e insertándolos en la posición correcta. Eficiente para listas pequeñas o parcialmente ordenadas.


Merge Sort

Divide la lista en mitades, ordena cada mitad y luego combina las mitades ordenadas para obtener la lista final ordenada. Es eficiente para listas grandes y garantiza un rendimiento O(nlog⁡n)O(nlogn) en todos los casos.


Quick Sort

Selecciona un "pivote" y particiona la lista en dos: elementos menores que el pivote y elementos mayores que el pivote. Luego, aplica el mismo proceso a cada sublista. Es eficiente para listas grandes y tiene un rendimiento promedio O(nlog⁡n)O(nlogn).


Heap Sort

Construye un heap (estructura de datos en forma de árbol) y luego extrae repetidamente el elemento máximo (en el caso de un max-heap) para obtener la lista ordenada. Tiene un rendimiento O(nlog⁡n)O(nlogn) y es in-place, lo que significa que no requiere almacenamiento adicional.


Radix Sort

Ordena los elementos basándose en sus dígitos, comenzando desde el dígito menos significativo hasta el más significativo o viceversa. Es eficiente para ordenar números enteros con un número limitado de dígitos.
